# Images Directory

Place your book cover and other images here.

## Recommended Images

- `book-cover.jpg` or `book-cover.png` - Book cover image
- `author-photo.jpg` - Author headshot (optional)
- `og-image.jpg` - Open Graph image for social media sharing (1200x630px recommended)

## Image Optimization Tips

- Use JPEG for photographs
- Use PNG for images with transparency or text
- Optimize images before uploading (use tools like TinyPNG, ImageOptim, or Squoosh)
- Recommended max width: 2000px for retina displays
- Keep file sizes under 500KB when possible

## Usage in Content

Reference images in your Markdown files:

```markdown
![Book Cover](/images/book-cover.jpg)
```

Or in HTML/Nunjucks templates:

```html
<img src="/images/book-cover.jpg" alt="The Data Flow Map book cover">
```
