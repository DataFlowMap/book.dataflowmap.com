# Quick Start Checklist

Use this checklist to get your book website up and running.

## Initial Setup

- [ ] Clone the repository to your local machine
- [ ] Install Node.js 18 or higher
- [ ] Run `npm install` to install dependencies
- [ ] Run `npm start` to test locally
- [ ] Visit `http://localhost:8080` to preview

## Customization

- [ ] Add book cover image to `src/images/book-cover.jpg`
- [ ] Add author photo to `src/images/author-photo.jpg` (optional)
- [ ] Update purchase links in `src/_data/site.json`
- [ ] Review and customize content in all `.md` files
- [ ] Adjust colors in `src/css/main.css` if desired

## Content Review

- [ ] Review home page (`src/index.md`)
- [ ] Review about page (`src/about.md`)
- [ ] Review author page (`src/author.md`)
- [ ] Review buy page (`src/buy.md`)
- [ ] Review resources page (`src/resources.md`)

## Pre-Deployment

- [ ] Test all internal links
- [ ] Test all external links (Amazon, Springer)
- [ ] Check mobile responsiveness
- [ ] Test in multiple browsers
- [ ] Verify all images load correctly
- [ ] Run `npm run build` successfully

## Deployment to Netlify

- [ ] Create Netlify account
- [ ] Connect GitHub repository
- [ ] Verify build settings
- [ ] Deploy site
- [ ] Set up custom domain `book.dataflowmap.com`
- [ ] Configure DNS (CNAME or Netlify DNS)
- [ ] Enable HTTPS
- [ ] Test live site

## Post-Deployment

- [ ] Verify all pages load correctly
- [ ] Test purchase links
- [ ] Check mobile experience
- [ ] Share with colleagues for feedback
- [ ] Set up analytics (Google Analytics or Netlify Analytics)

## Optional Enhancements

- [ ] Add email signup form
- [ ] Create downloadable sample chapter PDF
- [ ] Add testimonials/reviews section
- [ ] Create blog section for related content
- [ ] Add social media share buttons
- [ ] Implement search functionality
- [ ] Add FAQ section

## Maintenance

- [ ] Keep dependencies updated (`npm update`)
- [ ] Monitor site analytics
- [ ] Update content as needed
- [ ] Respond to reader feedback
- [ ] Add new resources over time
